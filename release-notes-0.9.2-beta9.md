## 本次更新

集中修复修复工具中的配置安全、错误恢复和存储占用问题。

### 修复与优化

- Node.js 内存设置不再覆盖用户已有的其他 Node 启动参数。
- 内存配置只读取经过校验的固定数值，不再作为 shell 配置直接执行。
- 主题重置必须唯一识别用户设置文件，多用户情况下会停止并提示，避免改错账号。
- npm 依赖安装失败时先隔离失败目录，再恢复原依赖，并明确显示恢复结果。
- 上传限制只保留当前有效的修改前备份，减少长期使用后的空间占用。
- 增加脚本回归检查，降低后续更新引入 Termux 脚本语法错误的风险。

### 说明

- 这是测试版。
- 多用户酒馆暂不提供自动主题重置，请根据候选路径确认用户后再处理。
