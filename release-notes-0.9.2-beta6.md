## 本次更新

新增一组面向常见启动故障的安全修复工具，减少手动进入 Termux 改文件的步骤。

### 新增与优化

- 新增“修复 npm 依赖”：重新安装前会先保留原依赖目录，失败时尽量自动恢复。
- 新增“重置网页主题”：主题导致网页打不开时，可恢复为 Dark Lite，并保留原设置文件副本。
- 新增 Node.js 内存上限设置：可按当前酒馆实例选择 2GB、4GB 或 6GB，不直接修改启动脚本。
- 修复操作会检查酒馆是否仍在运行，并在执行前显示影响说明和确认提示。

### 说明

- 这是测试版，建议先备份重要数据。
- 主题重置只会修改能够安全识别的设置文件；找不到兼容设置时不会强行改动。
- 内存上限并不代表设备一定能提供对应内存，低内存手机不建议选择过高数值。
